Analyseur-syntaxique-2013 - Resume du projet

Projet d'exercice C (niveau debutant/intermediaire) regroupant des utilitaires de
lecture/ecriture de fichiers et de manipulation de chaines de caracteres.

Structure du projet :
- io_utils.c / io_utils.h : Module d'entrees/sorties (ouverture, fermeture fichiers,
  recuperation d'arguments CLI)
- UtilString.c / util_string.h : Module de manipulation de chaines (calcul longueur,
  affichage, concatenation, extraction sous-chaine)
- Test1.c : Squelette minimal testant l'ouverture d'un fichier
- Test1.1.c : Lecture d'un fichier passe en argument, affichage et comptage des lignes
- Test1.2.c : Saisie stdin vers fichier resultat.txt jusqu'au mot-cle "FIN"
- Test2.c : Demonstration des fonctions de manipulation de chaines
- tests/tests.js : Tests unitaires JavaScript simulant les fonctions C
- Sauvegarde.txt : Fichier de test (donnees d'essai)
- A_faire.md : Liste de bugs connus et ameliorations suggerees
- README.md : Documentation du projet

Technologies :
- Langage C (C99/C11)
- Compilateur GCC >= 4.8
- Bibliotheques standard (stdio.h, stdlib.h, string.h)

Fonctions principales :
- ouvrir_fichier() / fermer_fichier() : Gestion securisee des fichiers
- verif_et_recup_arg() : Validation et recuperation d'arguments CLI
- calcLength() : Calcul de longueur d'une chaine
- aff() : Affichage d'une chaine
- copieFinChaine() / copieDebutChaine() : Concatenation de chaines
- coupeStr() : Extraction de sous-chaine

Bugs documentes (A_faire.md) :
- Allocation memoire sans +1 pour le '\0' (corrige dans UtilString.c actuel)
- Pointeurs non alloues dans main de Test2.c (corrige)
- Valeur de retour fgets ignoree dans Test1.2.c (corrige)
- strcpy sans verification de taille (corrige avec strncpy)
